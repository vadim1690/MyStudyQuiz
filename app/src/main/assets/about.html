My Study Quiz

 

  • Creation of a limited amount of tests about 5 at the same time.
  • Questions can be added to each test - a limit of 10 questions for free.
  • Each question can be of different types of American, true/false, and open question - there is no limitation in the free version.
  • You can edit the tests - delete them or add or delete questions.
  • You can edit the questions.
  • Custom settings options such as whether to display the answers immediately after answering or only at the end of the test, whether to record a score at the end, etc... with advertisements
  • There is no limit to the number of tests.
  • There is no limit to the number of questions.
  • A collection of pre-prepared famous tests of general knowledge. (can be deleted if you don't want to).
  • Option to add a timer for tests.
  • No ads.
  • You can see statistics on the tests (which questions are the hardest/easiest, how many times I got a passing grade, how many times I failed, etc.)
